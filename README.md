# Linyc Blog Pro

一个沉浸式博客演示项目，包含：

- Three.js 跑车开场动画（风阻流线、镜头切换、点击加速）
- 模型自动轮轴识别旋转
- 低干扰交互说明 HUD（`H` 常驻）
- 镜头名左下角淡入淡出字幕
- 博客页面：`首页 / 归档 / 分类 / 标签 / 关于 / 搜索 / 详情页`
- 深色模式、进度条、回到顶部
